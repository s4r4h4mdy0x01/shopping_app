# Technologies

 • Dart and Flutter 
 
 • Dio and Retrofit for RESTful API calls
 
 • Clean Architecture for a solid structure.
 
 • Bloc (Cubit) for state management.
 
 • injectable for dependency injection.
 
 • FlutterScreenUtil for responsive design
